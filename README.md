# stark-portal
A simple web portal with user authentication using JWT tokens.

## Overview
The stark-portal project is a basic web portal that demonstrates user authentication using JWT tokens. It allows users to log in as either an admin or a guest and provides access to a dashboard based on the user's role.

## Features
- User authentication with JWT tokens
- Admin and guest login functionality
- Role-based access to the dashboard

## Tech Stack / Built With
- HTML
- Python
- Flask
- JWT

## Installation & Setup
```bash
git clone https://github.com/CodeName-R4M/stark-portal.git
cd stark-portal
pip install -r requirements.txt
python app.py
```

## Usage
```python
# Run the Flask app
python app.py
```
